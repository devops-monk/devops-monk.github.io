How to Enable SSH on Ubuntu ?

SSH

Secure Shell (SSH) is a protocol used to securely log onto remote machines. It is the most common way to access remote Linux servers. 1. Enabling SSH on Ubuntu The SSH server is not installed by default on Ubuntu desktop systems but it can be easily installed from the standard Ubuntu repositories. To install and enable SSH on your Ubuntu system complete the following steps: sudo apt-get install openssh-server Once the installation is completed, the SSH service will start automatically.
